<h1>Python Calculator</h1>

<p>
A lightweight desktop calculator built with Python and Tkinter. The code is kept small and readable so you can easily explore it, change the layout, tweak the colors, or extend it with your own ideas. There are no external dependencies, so you can clone the project and run it right away.
</p>

<h2>Features</h2>
<ul>
  <li>Basic arithmetic operations</li>
  <li>Minimal, distraction-free interface</li>
  <li>Simple structure for quick customization</li>
  <li>Runs on Windows, macOS, and Linux</li>
</ul>

<h2>How to Run</h2>
<p>From the project root:</p>
<pre><code>python src/calculator.py</code></pre>

<h2>Project Structure</h2>

│

│

</code></pre>

<h2>Customization</h2>
<p>
You can change the window title, colors, fonts, and button layout directly in the Tkinter code. The project is intentionally small so you can experiment freely and turn it into your own version of a desktop calculator.
</p>
